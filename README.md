# Telematic Applications Project
This is the reporitory for the final project of Telematic Applications.

In this project via API requests, you get the information about the Barcelona Bicing parking slots (id of the slot, if it is free/reserved/occupied).
Also there is a system to generate local accounts and from these make reservations for slots.

Finally there is an option to see the quality of the air in the zone of the parking slot.
